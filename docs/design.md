# voltech.ai Design Guidelines

These guidelines must be strictly followed when creating or modifying any visual elements, components, or screens across all portals.

## 1. Core Visual Elements

| Element           | Specification                                                                            |
| :---------------- | :--------------------------------------------------------------------------------------- |
| **Primary Color** | `#00A676` (Sustainability Green)                                                         |
| **Accent Colors** | - Soft mint for panels<br>- Neutral greys for cards<br>- Optional warm accent for alerts |
| **Typography**    | - Display: `Playfair Display` font for headings<br>- Body: Clean `Inter` font for body   |

## 2. Supported Languages & Locales

*   **Malaysia** (`voltech.ai/my`)
    *   English (UK) - *Default*
    *   Bahasa Malaysia
*   **Indonesia** (`voltech.ai/id`)
    *   English (UK) - *Default*
    *   Bahasa Indonesia

## 3. UI/UX Principles

| Principle         | Description                                                 |
| :---------------- | :---------------------------------------------------------- |
| **Mobile-First**  | Optimised for 375px+ screens and low-bandwidth scenarios.   |
| **Glassmorphism** | Subtle glass effects in headers/overlays.                   |
| **Gradients**     | Earth-themed gradients for hero and impact sections.        |
| **Motion**        | Soft transitions, hover-lift on cards, progress animations. |
